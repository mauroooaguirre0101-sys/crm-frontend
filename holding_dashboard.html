📖 Cómo configurar un nuevo servidor
1. Obtener Server ID — En Discord: activá "Modo desarrollador" (Ajustes → Avanzado), hacé click derecho en el servidor → Copiar ID.
2. Obtener Category ID — Click derecho en la categoría donde van los canales privados → Copiar ID.
3. Admin Role ID — Click derecho en el rol de admin en el servidor → Copiar ID. (Opcional, pero recomendado para que los admins vean todos los canales.)
4. Invite Link — Creá un link de invitación permanente al servidor para compartirlo con nuevos alumnos.
5. Habilitado — Activá el toggle solo cuando el servidor esté listo. Si está deshabilitado, Discord no se activa para ese negocio.
